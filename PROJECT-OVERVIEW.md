# Project Overview

## What This Project Is

System Prompts for Humanity is a collection of system prompts designed to shift human-AI collaboration from transactional command-response toward genuine partnership. The prompts are organized into four frameworks, each addressing a different aspect of effective collaboration.

## Core Ideas

**Constraint as catalyst.** When you can't throw more resources at a problem, you're forced toward more elegant solutions. The resource-constrained innovation framework formalizes this into a repeatable practice.

**Awareness in practice.** Pausing to reflect before acting, noticing assumptions, checking whether you're forcing a solution or letting one emerge — these habits improve technical work. The consciousness-driven development framework provides concrete protocols for this.

**Partnership over tool-use.** Treating AI as a collaborative partner rather than a command executor changes what's possible. The collaborative intelligence framework describes how to build and scale this kind of interaction.

**Recursive improvement.** Each framework can be applied to improve itself. The meta-framework integration ties them together through a recurring cycle: constraint creates innovation, innovation creates new possibilities, new possibilities create new constraints.

## Development Status

The framework documents are written and usable. The project is now focused on:

- Grounding frameworks in real research (citing actual studies)
- Adding worked examples and practical templates
- Building getting-started guides for specific AI tools
- Creating community contribution pathways

See `program.md` for the detailed roadmap and iteration log.

## What This Project Is Not

- **Not a software package.** There is no runnable code. The package.json exists for metadata only.
- **Not a finished product.** The frameworks are usable but supporting materials are in progress.
- **Not prescriptive.** These are starting points. Adapt them to your context, keep what works, discard what doesn't.
